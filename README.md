KnockOutTweety
==============

Demo showing Knockout.js with Twitter &amp; Maps API using Play! Framework

### screenshot

## Requirements:
 - Play! Framework 1.2.4 (www.playframework.org)

## Running the demo:
 - git clone <this repo> 
 - cd KnockOutTweety
 - play test

Made for MetaRefresh (Bangalore) April 2012
